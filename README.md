Запуск тестов

Откройте терминал и выполните следующую команду:

Пример:

Для запуска тестов локально

```
gradle clean test -Dhost=local
```
Для запуска тестов даленно

```
gradle clean test -Dhost=remote
```

Для выбора браузера при тестировании необходимо добавить параметр:

```
-Dbrowser=chrome
```

Доступные браузеры для теста:

```
chrome
firefox
```
